CARLA 0.9.11 中的交通信号灯检测与自主控制
=========================================
本项目使用 CARLA 模拟器（版本 0.9.11）实现了一个简单的自主控制系统。该系统通过摄像头数据检测交通信号灯，并根据信号灯状态控制车辆在红灯时停车、绿灯时通行。

## 功能特性

1.交通信号灯检测：使用 RGB 摄像头和边界框进行检测

2.停走逻辑：车辆在红灯时停车，绿灯时恢复行驶

3.自主控制：根据交通信号灯状态自动调节油门和刹车

## 使用的技术

模拟器：CARLA Simulator 0.9.11
语言：Python 3
依赖库：
pygame – 用于渲染仿真窗口
cv2 (OpenCV) – 用于图像处理（如需扩展功能）
numpy、time、math、random – 用于计算和计时逻辑

1. 安装所需依赖包

```
pip install pygame opencv-python numpy
```

2.使用方法：

```
CarlaUE4.exe    # Windows 系统
```

3. 启动程序
```
python3 main.py
```
