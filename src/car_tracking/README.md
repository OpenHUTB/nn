# Carla-LowQuality-Viewer
CARLA 0.9.14 低画质版专用可视化脚本，针对低配置设备优化，适配低画质 CARLA 的 API 差异，解决着色器崩溃、异常类找不到等问题，实现轻量化的摄像头画面可视化+车辆自动识别交通信号灯（红灯停、绿灯行）与自动行驶功能。

## 功能说明
### 核心功能
1. **2D低画质可视化**：针对低配置设备优化，适配CARLA 0.9.14低画质版的API差异，解决着色器崩溃、异常类找不到等问题。
2. **交通信号灯识别**：车辆可自动识别交通信号灯状态，实现红灯停、绿灯行的逻辑。
3. **车辆自动行驶**：结合信号灯识别功能，完成轻量化的自动行驶控制。

### 运行命令
