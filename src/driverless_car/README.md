无人机图像分类深度学习 Demo
该项目是一个无需无人机硬件和仿真软件的无人机深度学习入门示例，以无人机图像分类为核心方向，使用 CIFAR-10 公开数据集模拟无人机航拍图像，搭建轻量化卷积神经网络（CNN）实现图像分类，并通过 Matplotlib 实现数据可视化和实时训练界面展示。
项目简介
核心功能
数据模拟：使用 CIFAR-10 数据集（包含飞机、汽车、鸟等 10 类常见物体）模拟无人机航拍采集的图像数据。
模型训练：搭建轻量化 CNN 模型，完成图像分类的训练与预测，适配 CPU/GPU 自动运行。
可视化展示：
展示无人机采集的图像样本
实时绘制训练损失 / 准确率曲线
展示模型预测结果（标注正确 / 错误）
兼容性优化：解决 Matplotlib 字体缺失、后端兼容报错等问题，可直接在 Python/PyCharm 环境运行。
技术栈
深度学习框架：PyTorch
数据处理：torchvision（数据集加载、数据变换）
可视化：Matplotlib、NumPy

代码结构说明
模块	功能描述
环境配置与全局参数	导入依赖库、解决 Matplotlib 兼容问题、定义数据集类别名称
数据预处理与加载	图像归一化、张量转换、批量加载 CIFAR-10 训练 / 测试集
轻量化 CNN 模型搭建	构建卷积层 + 池化层 + 全连接层的分类模型，加入 Dropout 防止过拟合
可视化函数	3 个核心函数：展示样本、绘制训练曲线、展示预测结果
模型训练与评估	实现深度学习经典训练流程（前向传播→损失计算→反向传播→参数更新），实时可视化
主程序入口	调用训练函数，设置训练轮数
核心模型结构
轻量化 CNN 模型包含以下层结构：
卷积层：3 层卷积（3→32→64→128 通道），提取图像特征。
池化层：最大池化（2×2），降低特征维度，减少计算量。
全连接层：2 层全连接（128×4×4→512→10），输出 10 类预测结果。
激活函数：ReLU，引入非线性，提升模型表达能力。
Dropout：0.5 概率丢弃神经元，防止过拟合。
可视化界面说明
界面名称	展示内容
无人机图像样本窗口	12 张 CIFAR-10 图像及对应类别，模拟无人机采集的图像数据
训练曲线窗口	左图：训练损失变化趋势；右图：训练 / 测试准确率变化趋势（实时更新）
模型预测结果窗口	16 张测试集图像，标注真实标签和预测标签（绿色 = 正确，红色 = 错误）
参数调整建议
训练轮数（epochs）：默认设置为 2，可在train_model(epochs=2)中调整，轮数越多训练效果越好，但耗时更长。
批次大小（batch_size）：默认 64，可在DataLoader中调整，批次越大训练速度越快，需占用更多内存。
学习率（lr）：默认 0.001，可在optim.Adam中调整，学习率过小训练缓慢，过大可能导致模型不收敛。

常见问题解决
数据集下载缓慢：可手动下载 CIFAR-10 数据集，放入./data目录，程序会自动识别。
可视化窗口闪退：代码已添加plt.pause(0.1)和plt.show(block=True)解决，若仍出现问题，可检查 Matplotlib 后端配置。
GPU 未启用：确认安装了 CUDA 版本的 PyTorch，且显卡支持 CUDA 加速。
