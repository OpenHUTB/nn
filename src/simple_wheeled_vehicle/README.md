# 简易轮式小车 MuJoCo 仿真模型
一款轻量化的**2驱动轮+1万向轮**轮式小车 MuJoCo 模型，适用于移动机器人运动控制入门学习。

## 模型概述
- **结构组成**：长方体底盘 + 2个后置驱动轮 + 1个前置万向轮
- **关节类型**：驱动轮采用铰链关节（沿Y轴旋转），万向轮采用球关节（360°自由转向）
- **控制方式**：驱动轮速度控制，支持差速转向
- **引擎兼容性**：MuJoCo 2.1 及以上版本

## 核心组件说明
| 组件 | 类型 | 功能 |
|-----------|------|----------|
| 底盘 | 长方体几何 | 小车主体，承载所有部件 |
| 左/右驱动轮 | 圆柱体几何 | 由速度执行器驱动，提供动力 |
| 万向轮 | 球体几何 | 支撑车身，实现自由转向 |
| 铰链关节 | 铰链类型 | 约束驱动轮沿固定轴旋转 |
| 球关节 | 球型类型 | 允许万向轮无驱动自由转向 |
| 速度执行器 | 速度类型 | 控制驱动轮旋转速度 |

## 快速上手
1. 将 `car_model.xml` 放入你的 MuJoCo 项目目录
2. 在 MuJoCo 模拟器或自定义代码中加载该模型
3. 通过为 `left_wheel_vel` 和 `right_wheel_vel` 执行器设置目标速度来控制小车

## 控制逻辑
- **直线运动**：为左右驱动轮设置相同的目标速度（正数前进，负数后退）
- **差速转向**：为两个驱动轮设置不同的目标速度（速度差越大，转向半径越小）

## 自定义优化技巧
- 调整底盘几何的 `size` 参数，修改车身尺寸
- 修改惯性参数的 `mass` 值，调整车身重量和动力学性能
- 更改速度执行器的 `ctrlrange` 参数，限制驱动轮最大转速
