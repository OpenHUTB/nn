

# 循环神经网络(RNN)
循环神经网络（Recurrent Neural Network, RNN）是一种专门处理序列数据（如时间序列、文本、语音）的神经网络，其核心特点是记忆能力——能够利用“历史信息”影响当前输出。


## 问题描述：

利用循环神经网络，实现唐诗生成任务。




## 数据集: 

唐诗





## 题目要求：
