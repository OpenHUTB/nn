#交通信号灯检测和自主控制在卡拉0.9.11

本项目使用CARLA模拟器（版本0.9.11）实现了一个简单的自主控制系统。系统利用摄像头数据检测**交通灯**，并控制车辆**红灯**，**绿灯**。

# #特点:

-  **红绿灯检测**使用RGB相机和边界框。
-  **停车和通行逻辑**：车辆在红灯时停车，绿灯时继续行驶。
-  **自动控制**：根据红绿灯状态自动调节油门和刹车。

## 技术的使用

- **模拟器**:[CARLA模拟器0.9.11]
(https://github.com/carla-simulator/carla)
- **语言**: Python 3
- **库**:
  - `pygame` – 用于渲染模拟窗口
  - `cv2` (OpenCV) – 用于图像处理（如果扩展）
  - `numpy`, `time`, `math`, `random` – 用于计算和时序逻辑


## 它是如何工作的

1. **传感器设置**：车辆配备前置RGB摄像头和其他传感器在卡拉。
2. **交通信号灯检测**:
   - 脚本使用CARLA的内置环境信息（边界框）检测红绿灯。
   - 决定灯的状态（红/绿）。
3. **控制逻辑**:
   - 如果前方的交通灯是红色的，车辆会刹车并停车。
   - 如果交通灯是绿色或没有相关的红灯，车辆继续行驶。
