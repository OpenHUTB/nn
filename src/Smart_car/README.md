# 无人车（自动驾驶汽车）

## 简介

无人车是一种能够自动感知周围环境并自主行驶的智能汽车，无需人工驾驶员操作。

## 技术等级

| 等级 | 名称 | 描述 |
|------|------|------|
| L0 | 无自动化 | 传统汽车，完全由人驾驶 |
| L1 | 驾驶辅助 | 自适应巡航、车道保持等功能 |
| L2 | 部分自动化 | 同时控制转向和加减速 |
| L3 | 有条件自动化 | 特定情况下自动驾驶 |
| L4 | 高度自动化 | 特定区域完全自动驾驶 |
| L5 | 完全自动化 | 所有情况下完全自动驾驶 |

## 核心技术

### 感知技术
**摄像头**：识别交通标志、车道线
**激光雷达**：3D环境扫描
**雷达**：检测车辆和障碍物
**超声波**：近距离检测

### 决策技术
**人工智能**：识别和决策算法
**路径规划**：规划行驶路线
**传感器融合**：整合多传感器数据

### 控制技术
**自动转向**：精确转向控制
**自动制动**：紧急制动系统
**自动加速**：智能速度控制

## 主要应用

### 乘用车领域
**特斯拉** - FSD系统
**小鹏汽车** - NGP辅助驾驶
**蔚来汽车** - NAD系统

### 商用车领域
**自动驾驶卡车**：货运物流
**无人公交车**：公共交通
**配送机器人**：最后一公里配送

### 特殊场景
**矿区作业车**：矿山运输
**港口AGV**：集装箱运输
**农业机械**：自动耕种

## 技术优势

✅ **提高安全性**：减少人为驾驶错误  
✅ **提升效率**：优化行驶路线  
✅ **减少拥堵**：车联网协调交通  
✅ **环保节能**：智能驾驶降低能耗  
✅ **便利出行**：解放驾驶员双手
