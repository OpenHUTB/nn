# 违规行为识别系统

## 项目简介
这是一个基于计算机视觉的违规行为识别系统，用于课堂或办公室场景，可以自动检测：
- 玩手机行为
- 睡觉/瞌睡行为  
- 长时间离席行为

## 功能特点
- ✅ 实时视频流处理
- ✅ 多行为检测
- ✅ 违规记录日志
- ✅ 可视化标注

## 技术栈
- Python 3.8+
- OpenCV
- Dlib (人脸关键点检测)
- NumPy
