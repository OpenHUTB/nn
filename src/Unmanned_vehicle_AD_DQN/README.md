# 基于DQN实现自动驾驶模型

这是深度 Q 网络 （DQN） 算法的实现，用于在 CARLA 中训练自动驾驶模型

## 环境配置

1. python版本3.7.9
2. 确保将carla在运行前配置到系统环境中

### carla环境配置

1. .egg文件的API请修改Environment.py文件中，删除位于import carla上方注释的#
2. .whl文件的API请在系统中下载对应文件，该部分的两种文件任选其一即可

## 使用步骤

确保carla在代码运行之前已经启动

### 训练
训练模型请运行：

```
Main.py
```
### 测试
测试训练后的模型，请运行：
```
Test.py
```
