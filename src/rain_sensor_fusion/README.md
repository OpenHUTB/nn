项目简介
rain_sensor_fusion：基于多传感器融合的自动驾驶雨天环境 3D 物体检测仿真项目。针对雨天雨雾遮挡、镜头沾水、激光雷达点云稀疏、视觉特征退化等问题，融合视觉相机、激光雷达 LiDAR、毫米波雷达多传感器数据，在 CARLA 仿真平台下实现复杂雨天场景的 3D 车辆、行人、障碍物检测模拟，提升自动驾驶感知系统在恶劣天气下的检测精度与鲁棒性。
项目核心功能
1. 基于 CARLA 模拟器搭建雨天仿真场景，复刻降雨、水雾、路面湿滑等天气效果；
2. 模拟车载多传感器部署：RGB 相机、激光雷达、毫米波雷达同步采集数据；
3. 多传感器数据配准与融合，弥补单一传感器在雨天环境下的感知缺陷；
4. 实现 3D 目标检测，输出障碍物 3D 框、类别、位置、尺寸、朝向信息；
5. 可视化检测结果、传感器点云、融合效果与雨天感知对比实验。
技术栈
• 仿真平台：CARLA
• 编程语言：Python
• 核心依赖：PyTorch、NumPy、OpenCV、Pygame
• 感知算法：多传感器数据融合、3D 点云处理、图像特征提取、3D 目标检测

项目创新点
专门针对雨天恶劣工况设计，贴合实际自动驾驶落地难点；
采用多传感器互补融合：相机补纹理、激光雷达补空间位置、毫米波雷达抗雨雾干扰；
基于 CARLA 全场景仿真，无需实车硬件，低成本完成算法验证与模拟实验；
可扩展不同降雨强度、不同车流密度，做对比消融实验。
