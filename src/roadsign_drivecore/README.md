# CARLA-SignPilot
基于CARLA的交通标志检测与智能车辆控制系统

---

## 1. 项目概述
CARLA-SignPilot 是一个基于CARLA仿真器构建的智能驾驶仿真项目，集成了交通标志检测（基于YOLOv8）、车辆手动/自动控制、红绿灯识别、碰撞保护等核心功能。本项目专为课程作业迭代开发，共分为10个版本，每个版本在前一版本基础上新增1个独立小功能，逐步实现智能驾驶系统的迭代与完善，满足课程作业分阶段提交的要求。

---

## 2. 版本迭代说明
本项目采用分版本迭代开发模式，共10个版本，每个版本仅新增1个小功能，确保每次作业提交的独立性、可操作性和完整性，清晰呈现项目的逐步优化过程。

| 版本（Version） | 新增功能（Added Function） | 功能说明（Description） |
| :--- | :--- | :--- |
| V1.0 | 基础显示功能 | 实现车辆生成、摄像头画面显示、CARLA视角跟随，为项目基础版本 |
| V2.0 | 手动驾驶控制 | 通过WASD键控制车辆前进、转向，空格键控制紧急刹车 |
| V3.0 | 红绿灯识别功能 | 检测红绿灯状态，遇到红灯时自动刹车，绿灯时恢复行驶 |
| V4.0 | 碰撞保护功能 | 添加碰撞传感器，检测到碰撞后自动重置车辆控制状态，避免车辆失控 |
| V5.0 | 手动倒车功能 | 在手动驾驶模式下，通过S键控制车辆倒车，提升手动操控完整性 |
| V6.0 | 限速标志控制功能 | 识别道路限速标志（30/40/50km/h），自动将车辆速度限制在对应数值范围内 |
| V7.0 | 车辆跟车巡航功能 | 检测与前车的距离，维持安全跟车距离（默认10米），前车过近自动减速，过远恢复巡航速度 |
| V8.0 | 手动急刹视觉反馈 | 手动模式下按空格键急刹时，界面左下角弹出红色闪烁提示，松开后自动消失，提升操作反馈 |
| V9.0 | 简易车道保持功能 | 基于摄像头画面识别车道线，计算车辆偏移量，轻微调整方向盘，实现自动驾驶模式下的车道居中 |
| V10.0 | 行驶数据记录功能 | 实时采集车辆速度、驾驶模式、红绿灯状态等数据，每1秒保存一条至本地TXT文件，便于后续分析 |

## 3. 运行环境配置
本项目依赖以下环境，需提前配置完成，确保项目正常运行：
- 操作系统：Windows 10/11（64位）
- CARLA仿真器：0.9.15版本
- Python环境：3.7.9版本
- 依赖库：`carla`、`pygame`、`numpy`、`ultralytics`（YOLOv8）、`opencv-python`
- 虚拟环境：推荐使用`carla_yolo37_env`（适配Python 3.7.9，避免环境冲突）

---

## 4. 运行步骤
1.  启动CARLA仿真器：双击打开CARLA安装目录下的`CarlaUE4.exe`，等待地图完全加载（出现城市画面即可）。
2.  激活虚拟环境：打开PyCharm，选择虚拟环境`carla_yolo37_env`，确保所有依赖库已安装完成。
3.  运行项目：打开项目主文件`Main.py`，点击运行按钮，即可启动项目，默认进入自动驾驶模式。

---

## 5. 操作说明
| 按键/操作 | 功能说明 |
| :--- | :--- |
| `M`键 | 切换手动驾驶/自动驾驶模式 |
| `W`键 | 手动模式：车辆前进 |
| `S`键 | 手动模式：车辆倒车 |
| `A`/`D`键 | 手动模式：车辆左/右转向 |
| 空格键 | 手动模式：紧急刹车 |
| `ESC`键 | 退出项目，自动清理所有仿真资源 |
| 窗口关闭按钮 | 关闭项目窗口，自动清理仿真资源，避免内存泄漏 |

---

## 6. 注意事项
- 运行项目前，必须先启动CARLA仿真器，否则会出现连接失败错误。
- 确保Python环境版本为3.7.9，依赖库版本与CARLA 0.9.15适配，避免版本冲突。
- 每次运行项目后，建议正常关闭窗口（ESC键或关闭按钮），确保资源完全清理，避免CARLA内存泄漏。
- 自动驾驶模式下，车辆会自动遵守红绿灯、限速标志，手动模式下需手动控制所有操作。
- 若出现画面黑屏、闪退等问题，可重启CARLA仿真器和项目，或检查摄像头配置是否正确。

---

## 7. 项目用途
本项目为课程作业迭代开发，主要用于展示智能驾驶系统的逐步构建过程，涵盖交通感知、车辆控制、模式切换等核心模块，可作为CARLA仿真、YOLO目标检测、智能驾驶控制相关课程的实践作业参考。
