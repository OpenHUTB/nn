一、项目简介

本项目针对轴距 300mm、整机质量 0.5kg 级的小型民用四旋翼无人机，基于 MuJoCo 2.3.7 物理仿真引擎构建高保真飞行动力学仿真平台，核心围绕适配该型无人机的串级 PID 姿态 - 位置控制算法展开开发、验证与参数优化。平台精准还原该级别无人机的旋翼升力 / 反扭矩特性、机身六自由度刚体动力学及低速飞行下的空气阻力效应，可复现室内无扰 / 0~3m/s 风场扰动环境下的悬停、1m/s 匀速直线轨迹跟踪场景，解决小型四旋翼实机测试成本高、参数调试风险大、周期长的问题，为该类无人机控制算法的快速迭代提供标准化验证环境。

二、功能特性 

高保真动力学仿真：无人机单旋翼升力与转速平方的正比关系、旋翼反扭矩耦合效应，空气阻力系数适配小型无人机低速（＜5m/s）飞行特性；

串级 PID 精准控制：采用 100Hz 控制频率的姿态环（横滚/俯仰/偏航）+ 位置环串级 PID 控制，实现 ±0.1m 精度的悬停、±0.15m 精度的 1m/s 匀速直线轨迹跟踪；

可控环境扰动：配置 0~3m/s 可调风速的均匀风场，模拟室内气流扰动对小型无人机飞行稳定性的影响；

量化性能评估：实时计算位置跟踪均方误差（RMSE）、姿态角波动范围、旋翼转速超调量等核心指标，输出包含 “是否达标” 结论的标准化评估报告；

轻量化可视化调试：通过 MuJoCo 原生可视化界面，实时查看无人机位姿、旋翼转速、控制指令变化，支持仿真暂停/步进调试；

三、环境要求（精准适配）​
1. 硬件要求（最低/推荐）​
处理器：最低配置 Intel i3-8100/Ryzen 3；推荐配置 Intel i5-10400/Ryzen 5​
内存：最低配置 8GB DDR4；推荐配置 16GB DDR4​
显卡：最低配置 集成显卡；推荐配置 NVIDIA GTX 1650（4G）​
存储：最低配置 5GB 空闲空间；推荐配置 10GB 空闲空间​
系统架构：最低配置 64 位操作系统；推荐配置 64 位操作系统​
2. 软件要求（精准版本）​
操作系统：版本要求 Ubuntu 20.04 LTS，必要性 强制，适配说明 优先选择 Linux 系统，Windows 需额外安装 VC++ 运行库​
Python：版本要求 3.9.7，必要性 强制，适配说明 经测试与 MuJoCo 2.3.7 完全兼容​
MuJoCo：版本要求 2.3.7，必要性 强制，适配说明 仅适配该版本，避免版本兼容问题​
numpy：版本要求 1.23.5，必要性 强制，适配说明 满足小型无人机动力学计算精度需求​
matplotlib：版本要求 3.6.2，必要性 强制，适配说明 适配数据可视化输出格式​
scipy：版本要求 1.9.3，必要性 强制，适配说明 支撑 PID 控制算法的数值计算​
PyTorch：版本要求 1.12.1，必要性 可选，适配说明 仅用于 PID 参数自动寻优功能
