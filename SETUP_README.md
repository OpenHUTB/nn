# Setup.bat 进度显示改进

## 改进概述

为 `setup.bat` 添加了实时进度显示功能，提供更清晰的安装过程反馈。

## 技术改进

### 1. 进度跟踪系统
- **11 个主要步骤**的自动编号和百分比计算
- **实时进度条**（50 字符宽度，使用 `#` 显示完成度）
- **步骤计数器**追踪当前执行进度

### 2. 计时功能
- **高精度计时器**：记录脚本开始时间和每步耗时
- **已用时间显示**：显示当前已用时间（分:秒格式）
- **预估剩余时间**：基于历史耗时平均值，智能预测完成时间

### 3. 状态输出系统
- **步骤结果函数** (`:step_result`)：标准化输出格式
  - `[OK]` - 步骤成功完成
  - `[SKIP]` - 步骤跳过（文件已存在）
  - `[DOWNLOAD]` - 下载新文件
  - `[ERROR]` - 步骤失败
- **清晰的视觉分隔**：使用分隔线区分每个步骤

### 4. 统计汇总
脚本结束时显示：
- 总步骤数 / 完成步骤数
- 总耗时
- 跳过的文件数
- 下载的文件数
- 错误数

## 技术实现细节

### 延迟变量扩展
使用 `enabledelayedexpansion` 和 `!变量!` 语法实现动态变量更新

### 时间计算
```batch
for /f "tokens=1-3 delims=:." %%a in ("%TIME%") do (
    set /a "START_H=%%a, START_M=1%%b-100, START_S=1%%c-100"
)
```
将时间转换为总秒数进行计算

### 进度条生成
```batch
set /a "BAR_SIZE=PERCENT/2"
for /l %%i in (1,1,!BAR_SIZE!) do set "PROGRESS_BAR=!PROGRESS_BAR!#"
```
动态构建进度条字符串

### 函数调用
使用 `call :label` 语法实现函数式调用
- `call :show_progress "步骤说明"` - 显示进度
- `call :step_result "status" "消息"` - 显示步骤结果
- `call :show_summary` - 显示汇总信息
